=Microthrading subsystem=

==Introduction==

==Features==

* System is based on concept of cooperative multitasking
* Context is switched explicitly by executin Yield method
* Microthreads can run in main thread and classic TThread instances
* Microthreads have much lower resource requirements and max. count of created thread is limit only by available memory
* Scheduler measure running time of microthreads and provide usage information
* Supported platforms are 32-bit Windows and Linux
