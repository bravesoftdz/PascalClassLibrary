=UpdateChecker=

Allow users to check for new application updates. If new version is available then information windows is shown and use can proceed to download and install refered setup program.

==Features==

* Can download setup from http or filesystem location
* Can read latest version description file and determine newest version
* Show new version changes
* Allow run setup with elevated privileges and exit currently running program


* Require Lazarus 1.0 FPC 2.6.0
