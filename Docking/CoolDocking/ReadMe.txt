=CoolDocking=

Is form docking system for Lazarus IDE.

=Download=

You can download source from SVN repository:

http://svn.zdechov.net/svn/PascalClassLibrary/Docking/CoolDocking


=Features=

==Completeded features==

* List docking
* Tabbed docking
* Popup control menu
* Conjoin floating form
* Icons on tabs and grabbers
* Store and restore forms layout to file or stream
* Predefined customization form

==Planned features==

* Dock forms groups
* User defined form header styles
* Ability to lock and unlock docked controls for accidently undocking
